# Hospital Emergency Room Dashboard

## 📊 Project Overview
This Power BI dashboard provides comprehensive analytics for hospital emergency room operations, covering a 19-month period from April 2023 to October 2024. The dashboard tracks 9,216 unique patient visits and offers actionable insights to optimize ER performance and patient care.

## 🎯 Key Features

### Four Interactive Views:
1. **Monthly View** - Granular analysis of single-month performance
2. **Consolidated View** - Comprehensive 19-month trend analysis
3. **Patient Details** - Individual patient record exploration
4. **Key Takeaways** - Executive summary with critical insights

## 📈 Key Metrics Tracked

- **Patient Volume**: 9,216 total patients
- **Average Wait Time**: 35.3 minutes
- **Patient Satisfaction Score**: 4.99 out of 10
- **Referral Volume**: 3,816 patients referred
- **Admission Rate**: 50.04% admitted, 49.96% treated and released

### Patient Demographics
- **Age Distribution**: Adults (30-39 years) represent the largest group with 1,200 patients, followed by young adults (20-29 years) with 1,188 patients
- **Gender Split**: 51.05% Female, 48.69% Male, with some patients declining to identify
- **Race Distribution**: 
  - White: 2,571 patients (27.9%)
  - African American: 1,951 patients (21.2%)
  - Asian: 1,060 patients (11.5%)
  - Multi-racial: 1,557 patients (16.9%)

### Operational Performance
- **Peak Days**: Mondays (1,377 patients), Saturdays (1,322 patients), and Tuesdays (1,318 patients)
- **Busiest Hours**: 11 AM, 7 PM, 1 PM, and 11 PM
- **Patients Seen Within 30 Minutes**: 59.32% (5,000 patients within target)

### Department Referrals
- **Top Referrals**:
  1. General Practice: 1,840 cases
  2. Orthopedics: 995 cases
  3. Physiotherapy: 276 cases
  4. Cardiology: 248 cases
- **No Referral Needed**: 5,400 patients (58.6%)

### Wait Time Analysis
- Average wait time of 35.3 minutes indicates room for improvement in patient flow
- Satisfaction score of 4.99 suggests moderate patient experience with opportunities for enhancement

## 💡 Strategic Recommendations

1. **Staffing Optimization**: Increase staffing during peak hours (11 AM, 7 PM, 1 PM, 11 PM) and peak days (Monday-Tuesday)
2. **Wait Time Reduction**: Implement fast-track protocols to improve the 59.32% within-target rate
3. **Patient Satisfaction**: Focus on reducing wait times and improving communication to boost satisfaction scores
4. **Referral Management**: Strengthen partnerships with General Practice and Orthopedics departments given high referral volumes
5. **Weekend Capacity**: Ensure adequate weekend staffing to handle high Saturday volumes

## 🛠️ Tools & Technologies

- **Power BI Desktop** - Dashboard development
- **DAX** - Advanced calculations and measures
- **Power Query** - Data transformation and cleaning

### Visualizations Include:
- Line charts for trend analysis
- Donut charts for categorical distributions
- Horizontal bar charts for departmental comparisons
- Heatmaps for day/hour patient volume analysis
- KPI cards for key performance indicators
- Interactive slicers for date range selection

## 🚀 How to Use

1. **Monthly View**: Select specific year and month from dropdown filters
2. **Consolidated View**: Toggle date range slider for custom period analysis
3. **Patient Details**: Search and filter individual patient records
4. **Key Takeaways**: Review executive summary for strategic planning

## 📁 Data Structure

The dashboard analyzes patient records including:
- Patient demographics (age, gender, race)
- Admission dates and wait times
- Department referrals
- Admission status
- Satisfaction scores

## 🔐 Data Privacy

All patient names and IDs shown are anonymized. This dashboard complies with HIPAA privacy standards for healthcare data presentation.
